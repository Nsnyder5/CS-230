# CS-230
Reflection: Draw It or Lose It

The Gaming Room Client & Software Requirements: 

The Client: The client for this project is The Gaming Room, a company looking to expand the reach of their existing gaming services.

Software Design: The goal was to transition their popular Android-based game, "Draw It or Lose It," into a web-based application.Requirements: The primary requirement was to support users across multiple platforms (Windows, Mac, Mobile) while maintaining a single, consistent game state in server memory. This was essential to prevent duplicate games, players, or teams and to ensure unique naming conventions throughout the application.
